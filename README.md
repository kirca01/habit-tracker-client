# Habit Tracker Client

A React frontend for the Habit Tracker app — track your daily habits with streaks and a GitHub-style heatmap.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** — Build tool
- **Tailwind CSS v4** — Styling
- **Axios** — HTTP client
- **React Router** — Navigation

## Features

- Login and registration
- Dashboard with all habits
- Daily check-in with one click
- 🔥 Streak counter
- GitHub-style heatmap (last 6 months)
- Progress bar for daily completion
- Color-coded habits

## Getting Started

### Prerequisites

- Node.js 18+
- [Habit Tracker API](https://github.com/kirca01/habit-tracker-api) running locally

### Run

```bash
npm install
npm run dev
```

App will be available at `http://localhost:5173`
